Excerpt from HKUDS/LightRAG README:

> LightRAG: Simple and Fast Retrieval-Augmented Generation. [The project implements] a knowledge graph-based RAG system that performs dual-level retrieval combining entity/relation extraction with vector similarity search. [It] consistently outperforms baseline approaches (NaiveRAG, RQ-RAG, HyDE, GraphRAG) across multiple evaluation domains including agriculture, computer science, legal, and mixed datasets.

Source: https://github.com/HKUDS/LightRAG
