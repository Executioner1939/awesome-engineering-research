Excerpt from surrealdb.com/docs/surrealdb:

> a multi-model database built in Rust designed to unify multiple data models into a single, powerful engine.
>
> SurrealQL ... combines the best ideas from SQL, NoSQL and graph within a single native syntax.
>
> a single Rust binary that can run embedded (in-app), in the browser (via WebAssembly), in the edge, as single backend node, or in a distributed cluster.
>
> powerful search and retrieval (vector, full-text, hybrid) and blend[ing of] semantic, graph, and relational intelligence.
