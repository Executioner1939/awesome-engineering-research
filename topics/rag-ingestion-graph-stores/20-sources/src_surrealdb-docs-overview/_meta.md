# Meta

The canonical SurrealDB overview. Grounds four claims: (1) multi-model (document + graph + relational + time-series + geospatial + KV), not graph-first; (2) SurrealQL as a unified query language combining SQL, NoSQL, and graph syntaxes; (3) widest deployment surface in this comparison — embedded in-app, in-browser WASM, edge, single-node, distributed cluster; (4) "powerful search and retrieval (vector, full-text, hybrid)" framing with explicit semantic + graph + relational blend.
