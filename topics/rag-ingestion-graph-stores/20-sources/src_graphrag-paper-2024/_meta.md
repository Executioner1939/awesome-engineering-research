# Meta

The seminal Microsoft Research paper introducing GraphRAG. Latest arXiv version 2025-02-19. Grounds the canonical structural definition of graph-RAG: an LLM-built two-stage knowledge graph index plus pregenerated community summaries, used to answer "global" sensemaking questions over a corpus where flat vector retrieval fails.
