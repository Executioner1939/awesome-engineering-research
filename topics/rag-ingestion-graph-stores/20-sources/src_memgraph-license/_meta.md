# Meta

Memgraph Community ships under BSL 1.1 with an Additional Use Grant; Enterprise ships under the separate MEL (Memgraph Enterprise License), not captured in this run. BSL restrictions: no embedding to third parties, no DBaaS, no use to build competing products. The Change Date string in BSL.txt is malformed (`2030-15-05` is not a valid ISO date) so the exact day is ambiguous, but conversion-to-Apache-2.0 is explicit.
