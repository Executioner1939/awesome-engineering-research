# Meta

SurrealDB ships under BSL 1.1 with a SurrealDB-specific Additional Use Grant. Explicit Apache-2.0 conversion on 2030-01-01. Restriction: cannot use the Licensed Work to provide database functionality to third parties (i.e. cannot offer SurrealDB-as-a-Service without a commercial license). Explicitly "not an Open Source license" per its own text.
