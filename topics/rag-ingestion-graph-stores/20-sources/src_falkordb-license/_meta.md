# Meta

FalkorDB ships under the Server Side Public License v1 — the MongoDB-derived license that requires any third party offering FalkorDB as a managed service to release the full service source code. SSPL is the strictest of the three databases in this comparison. FalkorDB is **not** OSI-approved open source.
