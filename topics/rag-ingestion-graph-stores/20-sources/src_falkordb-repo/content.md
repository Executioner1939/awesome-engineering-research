Excerpt from github.com/FalkorDB/FalkorDB README:

> A super fast Graph Database uses GraphBLAS under the hood for its sparse adjacency matrix graph representation.
>
> Compatible with OpenCypher query language, including proprietary extensions for advanced querying capabilities.
>
> Our goal is to provide the best Knowledge Graph for LLM (GraphRAG).
>
> FalkorDB is the first queryable Property Graph database to leverage sparse matrices for representing the adjacency matrix in graphs and linear algebra for querying.

Source: https://github.com/FalkorDB/FalkorDB
