# Meta

The canonical FalkorDB README. Grounds three claims: (1) the engine is built on GraphBLAS sparse-adjacency matrices, (2) the query language is openCypher with proprietary extensions, (3) the project positions itself explicitly as "the best Knowledge Graph for LLM (GraphRAG)" — that is, the GraphRAG-workload narrative is on the front page, not a marketing afterthought.
