The `moon templates` command will list all templates available for [code generation](https://moonrepo.dev/docs/commands/generate).
This list will include the template title, description, default destination, where it's source files
are located, and more.

```shell
$ moon templates
```

### Options [​](https://moonrepo.dev/docs/commands/templates\#options "Direct link to Options")

- `--filter` \- Filter templates by a search term.
- `--json` \- Print templates in JSON format.

### Configuration [​](https://moonrepo.dev/docs/commands/templates\#configuration "Direct link to Configuration")

- [`generator`](https://moonrepo.dev/docs/config/workspace#generator) in `.moon/workspace.*`
